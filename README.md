# Módulo Notas Crédito y Débito - DIAN

Este proyecto permite generar y consultar notas crédito y débito electrónicas según el Anexo Técnico 1.9 de la DIAN.

## Requisitos

- Python 3.10+
- uv (https://github.com/astral-sh/uv)

## Instalación

```bash
uv venv .venv
source .venv/bin/activate  # En Windows: .venv\Scripts\activate
uv pip install -r requirements.txt
```

## Ejecución

```bash
python main.py
```
